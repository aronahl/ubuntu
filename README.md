# ubuntu
Expand base ubuntu images with tools that I usually need.
  * bsdmainutils
  * command-not-found
  * curl
  * jq
  * man
  * netmask
  * python-pip
  * socat
  * vim-nox
  * w3m

## usage

```bash
$ docker run --rm -it aronahl/ubuntu:14.04 
```

or

```bash
$ docker run --rm -it aronahl/ubuntu:16.04 
```
