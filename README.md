# ubuntu
Expand base ubuntu 16.04 image with tools that I usually need.
  * bsdmainutils
  * command-not-found
  * curl
  * git
  * htop
  * jq
  * less
  * man
  * mosh
  * netmask
  * openssh-client
  * patch
  * python-pip
  * socat
  * vim-nox (with gruvbox theme, airline, and nerdtree)
  * w3m

## usage

```bash
$ docker run --rm -it aronahl/ubuntu:14.04 
```

or

```bash
$ docker run --rm -it aronahl/ubuntu:16.04 
```
