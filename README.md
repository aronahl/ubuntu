# ubuntu
Expand base ubuntu image with tools that I usually need.

  * bsdmainutils
  * command-not-found
  * curl
  * git
  * htop
  * jq
  * less
  * man
  * mosh
  * netmask
  * openssh-client
  * patch
  * python-pip
  * socat
  * vim-nox
    * gruvbox
    * nerdtree
    * typescript-vim
    * vim-airline
    * vim-airline-themes
    * vim-fugitive
    * vim-regionsyntax
    * vim-signify
  * w3m

## usage

```bash
$ docker run --rm -it aronahl/ubuntu:14.04 
```

or

```bash
$ docker run --rm -it aronahl/ubuntu:16.04 
```
