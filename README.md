# ubuntu
Expand base ubuntu image with tools that I usually need.

  * bsdmainutils
  * command-not-found
  * curl
  * git
  * htop
  * jq
  * less
  * man
  * mosh
  * netmask
  * openssh-client
  * patch
  * python-pip
  * socat
  * vim-nox with goodies from [vimarsee](https://github.com/aronahl/vimaresee.git)
  * w3m

## usage

```bash
$ docker run --rm -it aronahl/ubuntu:14.04 
```

or

```bash
$ docker run --rm -it aronahl/ubuntu:16.04 
```
